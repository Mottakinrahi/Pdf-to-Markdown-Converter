# **Python Quick Reference Guide**

This document covers the basics of Python — a **high-level** , *interpreted* programming language. It
is designed to be ***simple and readable*** .

# **Key Features**

- Simple and readable syntax
- Dynamically typed
- Large standard library
- Cross-platform support

# **Getting Started**

Follow these steps to run your first Python script:
1. Install Python from python.org
2. Open a terminal or command prompt
3. Type python hello.py and press Enter

## **Hello World Example**

The simplest Python program is just one line: **print("Hello, World!")** . It calls the built-in print
function to output text to the terminal.

# **Why Use Python?**

Python is used across many fields — from **web development** and *data science* to automation and
artificial intelligence. Its clean syntax makes it a great first language and a powerful tool for
experienced developers.
